# Основы программирования на C++ и Математическое моделирование алгоритмов

Реализация фундаментальных алгоритмов обработки массивов, матричной алгебры и циклических структур на языке **C++**, совмещенная с графическим проектированием логики (блок-схемами).

## 📌 Описание программ и файлов

### 💻 Исходный код (C++)
* **`matrix-sort-rows.cpp`** — Программа для генерации двумерного массива (матрицы 5x5) случайными вещественными числами и последующей сортировки каждой строки по возрастанию.
* **`array-odd-squares-sum.cpp`** — Одномерный целочисленный массив из 20 элементов (диапазон значений от -50 до 50). Скрипт фильтрует нечётные элементы и вычисляет сумму их квадратов.
* **`array-digit-sum-transform.cpp`** — Алгоритм трансформации массива: принимает пользовательский набор чисел, заменяет каждое число на сумму его цифр, после чего выполняет поиск глобального минимума и максимума.

### 📊 Блок-схемы алгоритмов (Документация)
* **`flowchart-power-series-sum.docx`** — Графическое описание и формализация циклов `while` и `do-while` для расчета суммы членов степенной последовательности.
* **`flowchart-nested-loops-pattern.docx`** — Подробная блок-схема структуры вложенных циклов для генерации сложных условных матричных и строковых паттернов.

## 🛠️ Стек технологий
* **Язык программирования:** C++ (стандарт C++11 / C++14)
* **Инструменты:** Библиотеки `<vector>`, `<algorithm>`, `<iomanip>`, `<sstream>`
* **Проектирование:** Microsoft Word (графические примитивы блок-схем)
